Redmine:

Redmine is a flexible project management web application, which is used by Infra for task management, where it is(save it):
https://pdupc-redmine.mo.sw.ericsson.se/projects/sgsn-infra/issues

Portal:

A site integrates and automates delivery, build, regression and result displaying for SGSN-MME development

CIQ:

A tool helps to automate deliverying for CI process

Shade:

An IDE based on Eclipse with BT-Erl, ClearCase and Erlang plugins for SGSN-MME designers

Tecsas:

A light-weight SGSN-MME simulator which is used by both GTT and Simfast as SUT

T2:

Tecsomaten2 as it's full name, which is a distribution regression system for GTT regressions

Arrakis:

A distribution regression system for TTCN regressions based on Simfast

eXmachina:

A tool detects impacted GTT/TTCN test cases by analysing your Erlang code change, which helps to provide accurate regression scopes

Erietveld:

Code review site used by SGSN-MME developers

Infrabbit Hole:

A central site which keeps track of all tools used within SGSN-MME no matter it's from Infra or program, there you can find new tools or publish your new tools, also a nice place to find correct contact for tools(http://sgsn-services01.mo.sw.ericsson.se:1111/)
